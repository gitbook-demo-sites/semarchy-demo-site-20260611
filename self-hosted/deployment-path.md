# Deployment path

The deployment space should make install decisions easy to compare.

- Requirements and sizing

- Initial admin account setup

## Why this should be its own landing layer

The current docs have the material. A GitBook demo should show how to present it as a decision tree instead of a long hierarchy of install pages.
